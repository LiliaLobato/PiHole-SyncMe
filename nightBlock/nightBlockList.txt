# ==========================================================
# Night curfew list - blocked overnight (see schedule.json)
# One entry per line. '#' = comment.
#   - exact FQDN            -> exact deny   (e.g. www.instagram.com)
#   - contains regex chars  -> regex deny   (^ $ | ? * + ( ) [ ] { } \)
#   - force regex with      -> re:<pattern>
# Bare '.' and '-' do NOT make a line a regex.
#
# NOTE on apps vs browsers: a website (e.g. youtube.com) is only the
# front door. The mobile APP usually talks to a different API host and
# streams media from a CDN, so blocking just the ".com" stops the
# browser but not the app. Each service below lists the app/CDN hosts
# too. See the plan doc for the DoH/Private Relay caveat.
# ==========================================================

# --- Social / doomscroll ---

(\.|^)reddit\.com$
(\.|^)tiktok\.com$
(\.|^)facebook\.com$
(\.|^)fb\.com$
(\.|^)pinterest\.com$

# images / thumbnails / avatars
